# How to Make Your First Git Commit

## Overview

This guide explains how to track a file with Git for the first time.

**Time required**: 5 minutes  
**Prerequisites**: Git installed on your machine

---

## Steps

### 1. Create a file
touch README.md
### 2. Stage the file
git add README.md
### 3. Commit the file
git commit -m "Initial commit: add README.md"
### 4. Verify the commit
git log --oneline
You should see one line with your commit message.
