# Tutorial: Markdown Basics for Technical Writers

## What You Will Learn

By the end of this tutorial, you will be able to:

- Format text with headings, bold, and italics
- Create lists and code blocks
- Write a basic README in Markdown

**Time required**: 15 minutes

---

## 1. Headings

Use `#` to create headings.

```markdown
# Heading 1
## Heading 2
### Heading 3

## 2. Bold and Italics
**bold text**
*italic text*

## 3. Lists
- Item 1
- Item 2
- Item 3

## 4. Code Blocks
Use triple backticks to show code:

```bash
git init
```
